# Changelog

All notable changes to this project are documented in this file.

This project is currently in the `0.1.0-beta` release channel.

## [0.1.0-beta.1] - 2026-07-03

### Core
- Full database schema optimization with Prisma.
- Soft-delete implementation for critical business data.
- Multi-tenant data isolation (Farm-level filtering).
- Role-Based Access Control (RBAC) mechanism.

### Documentation
- Imported comprehensive enterprise documentation structure.
- Added bilingual support (README & README-vn).
- Added SECURITY.md, CONTRIBUTING.md, and CHANGELOG.md.

## [0.1.0-beta.0] - 2026-07-01

### Core
- Initial project bootstrap with Next.js 15 App Router.
- Supabase integration for Auth and Database.
- Base UI implementation using shadcn/ui.
- Pig, Barn, Inventory, Health, Breeding modules scaffold.

### Documentation
- Initial README.
